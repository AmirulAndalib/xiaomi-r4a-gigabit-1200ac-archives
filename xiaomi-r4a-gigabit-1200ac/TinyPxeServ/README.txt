Unbrick Xiaomi Router :

1) download the stock required firmware.

2) Download Pxeserv zip and unzip it to a specified folder.

3) Plug in Ethernet to pc from router.

4) Disable Wifi Adapter from control panel wifi settings.

5) go to Ethernet wifi settings from control panel and go to ipv4 options

Change settings like this below :

1) ip : 192.168.31.2

2) subnet : 255.255.255.0

3) gateway : 192.168.31.1

Then save these settings...


6) run the TinyPxeServ.exe from the specified extracted folder.

7) tap on httpd box (tick) then select firmware file from the below option.
